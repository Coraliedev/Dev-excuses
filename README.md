# dev-excuses

## Deployed on Vercel
https://dev-excuses-vue.vercel.app/

## Get the Code
- Clone Repo
- mkdir dev-excuses
- cd dev-excuses
- git clone https://github.com/Coraliedev/dev-excuses.git

## Run Server (API)
- cd backend
- npm install
- npm run start
 ### Run test API 
  npm run test
    
## Run Server (Client)
- cd frontend
- npm install
- npm run dev
